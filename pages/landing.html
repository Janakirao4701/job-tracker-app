How it works
Three steps. That's it.
No forms to fill. No copy-paste. Gemini AI does the extraction for you.
📋
Step 1
Copy the job description
Select all text on any job page — LinkedIn, Naukri, Indeed, company sites — and copy it (Ctrl+A, Ctrl+C).
⚡
Step 2
Click Extract & Save
Hit TRACK on the right edge of the page, then click ✦ Extract & Save. Gemini AI fills company, title, URL and JD instantly.
📊
Step 3
Track & Export
Update status as you progress through rounds. Export a professional Excel report anytime with all your data.

Setup Guide
Get running in 5 minutes
One-time setup. After this you just copy + click.
⚙️ One-time Setup
1
Install the extension
Download the extension folder and go to chrome://extensions → Enable Developer Mode → Load Unpacked → select the folder.
2
Get your free Gemini API key
Go to aistudio.google.com → Get API Key → Create API key. It's completely free with generous limits.
3
Paste key in the setup screen
Click the TRACK button on any page → the setup screen appears → paste your key → Save & Continue.
4
Create your account
Sign up with email and password. Your applications will sync to cloud so you can access them from any device.
5
Start tracking
Go to any job page, copy the JD, click TRACK → ✦ Extract & Save. Done in 2 seconds.
